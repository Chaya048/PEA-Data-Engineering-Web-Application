# PEA-Data-Engineering-Web-Application

- Developed a full-stack data application using Python (Streamlit) and Flask API to visualize electrical infrastructure data for the Provincial Electricity Authority (PEA).
- Deployed a Telegram Chatbot integrated with Generative AI (Gemini), allowing users to query the database
